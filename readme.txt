INSTALLATION INSTRUCTIONS:

Double-click pbpagectrl.dpk.  This will open the package in Delphi.

Click the Install button to install the run time package.  The PBPageControl component should appear on a palette called APHIControls.

You may also need to include the directory where you save this project in your Delphi Library path (Tools | Environment Options..., Library tab).
